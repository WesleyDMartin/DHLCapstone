The Project

This project evolved from Leigh Cassel's desire to create a unique experience which will allow students to learn about different cultures from the safety of their own classroom. Our goal is to provided a way to truly experience this learning in a Virtual Reality application. To be able to accomplish this unique experience, multiple different technologies have been brought together. To assure everybodyâ€™s safety, no vibranium was used to build this project.

With all jokes aside, the technologies that has been used to accomplish this project has been listed below.

  • The desktop application has been built on Unity where the virtual reality experience will be provided on oculus headset
  • The communication between the user and the application has been supported by TensorFlow and Googles speech to text API in the backend
  • The administration web portal has been built on rails for the backend and Vue for the front end

    •      The Story

    When we think about learning, we often imagine books, pencils, and paper; however, we now understand that learning happens in many different ways. Teachers know that interactions, relationships, and hands-on experiences are essential within the learning environment. The Digital Human Library is one of the first educational platforms to focus on learning through human connections. This Connection-based learning/education is an approach focused on building relationships and partnerships between students, teachers, families, experts and organizations. The DHL aims to "create a Cultural Learning Catelogue that allows users to interact with cultural experts.

    Creating an interactive catelogue means that students will obtain a deeper understanding of the many cultures that exist around the world. The founded of The Digital Human Library, Leigh Cassell, explains that "The need for cultural understanding and empathy is, and will continue to be, at the forefront of necessary learning in our increasingly globalized world." The Digital Human Library provides educators and students with a resource that uses first-hand expereinces to deliver information.

    Our project aimed to create a Virtual Reality experience where people are able to ask questions to the system or VR guide. The guide will be able to provide a text-based answer, along with video information, in a virtual reality format. Depending on availability, the viewer will be able to explore 360 videos as well as regular videos. This format will allow people to better understand connect and with people from various cultures. While the system currently incorporates only a few different cultures, it will continue to be updated to eventually allow users to vitrually travel the world.
