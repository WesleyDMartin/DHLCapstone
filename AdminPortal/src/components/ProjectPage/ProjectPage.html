The Project

This project evolved from the story where Leigh Cassel wanted to create a unique experience for the people which will allow them to learn about different cultures within the comfort of their own home. The experience will be provided in the virtual reality format. To be able to accomplish this unique experience, multiple technologies has been put together. With all jokes aside, the technologies that has been used to accomplish this project has been listed below.

  • The desktop application has been built on Unity where the virtual reality experience will be provided on oculus headset
  • The communication between the user and the application has been supported by TensorFlow and Googles speech to text API in the backend
  • The administration web portal has been built on rails for the backend and Vue for the front end

    •      The Story

    Accessing educational material and the quality of education still remains a challenge in different parts of the world. Digital Human Library addresses that by creating connections-based learning opportunities for teachers and students through variety of social innovation projects informed by research. Connection-based learning/education is an approach focuses on building relationships and partnerships between students, teachers, families, experts and organizations.

    When it comes to education, often time people focus on only academia. One area that often gets overlooked i s the culture. Canada being the melting pot of cultures where people from different ethnicity collaborates side by side, but do not know enough about each-others culture. This Project is here to address that educational side by creating a more personalized experience.

    This project has created a system where people will be able to ask questions to the system or VR guide and the guide will be able to provide a text-based answer along with video information in a virtual reality format. Depending on availability, the viewer will be able to explore 360 videos along with 2D videos. This will allow people to be able to connect and understand people from other culture from a broader perspective and allow people to have better cross-culture communication. The system currently supports only a few cultures at the moment. However, in the future, it will update to cover a lot more cultures that will allow people to explore the cultural horizon.
