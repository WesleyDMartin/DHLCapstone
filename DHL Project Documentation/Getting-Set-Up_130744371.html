    • Overview

    The main Digital Human Library project prototype consists of three main programs. These are:

    1. Unity project that includes VR interactivity,

    2. The Speech Recognizer, written in C#,

    3. Python script to match questions to pre-defined culture questions

    It is worth noting that the DHL prototype is not yet a production-ready application and may require some configuration of the user’s local machine to get up and running - these are outlined in the following sections.

    Get the project from GitHub

    The projects and source for the Digital Human Library are currently located on a private repository to which access may be granted upon request by the team. In order to pull the repository, the usual methods to clone a repository from GitHub can be used. For example, the GitHub Desktop application may be used to download the project files. An alternative would be to use the ‘git clone’ command in GitBash.

    Python Script

    In order to run the Python script, Python must be installed on the target Windows machine. Python may be downloaded from https://www.python.org/downloads/.

    The script file, service.py, requires the TensorFlow Python to be installed. Please install Python version 3.7.0 to ensure that the features utilized in the script run without error.

    After installing Python and attempting to run service.py, some modules may be missing. Install each missing module using the command ‘pip install <modulename>’.

    Speech Recognizer

    The Speech Recognizer should be run as a Visual Studio project. To open the project in Visual Studio 2019, go to File → Open → Project/Solution and navigate to DHL Capstone / Speech Recognizer / SpeechRecognizer.sln within the file system. Alternatively, double-click SpeechRecognizer.sln if Visual Studio is configured to automatically open the ‘.sln’ file extension.

    Unity VR App

    The VR application included in the project files handles the main interface for communicating with the avatar and getting video results back. In order to run the Unity project, the user must have the Unity game engine installed on their Windows machine.

    The official Unity site can be found at https://unity.com/ and a free version of the game engine is available at https://store.unity.com/ (under the 'Individuals' section).

    NuGet Packages

    Some NuGet packages may need to be installed before the Speech Recognizer or VR application can work. These packages will be indicated as missing upon running the project. To install the missing packages in Visual Studio, go to Project → Manage NuGet Packages, and then browse for the package indicated to be missing.
