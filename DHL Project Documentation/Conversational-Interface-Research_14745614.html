Summary

This article represents the effort put forth to research possible conversational interface options

TensorFlow

TensorFlow comes with a premade library which can be use to compare how similar two sentences are. This could be a perfect solution to our need to compare two sentences together, which we will need to do to figure out which question a user is asking.

This could also be used to cover more general questions, such as “how are you” or “what’s your name”.

This library is built into python, so we will need to ensure that TensorFlow can work with our project.

Watson

IBM’s Watson is another alternative, more work will need to be done.
