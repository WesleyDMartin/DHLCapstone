Summary

This article summarizes the research performed into the Amazon Web Services APIs, and how they will assist and work with our project.

AWS Rekognition is an image analysis and recognition software which is built on top of deep learning technology. This is an AI system which is used to analyze and recognize images. AWS recognition has all the deep learning technologies that is available without having to learn any of the deep learning technologies.

 

AWS recognition was released in 2016 which was the first cloud image analysis service by amazon. It makes image processing easier for developers. It can do face comparison, provide a description, can estimate the age of a person, detect objects like grass, tree, etc. It can also detect the text if there is any in an image.

For the DHL project, AWS is one of the platform that has been investigated as one of the potential solution that will aid users to identify an image, text, object etc. AWS is one of the largest AI platform that offers multiple solutions such as Deep learning, Natural language process, image recognition, etc. One of the beauty of using AWS’s platform is that the user will not have to learn machine learning or deep learning.

Research Details

For this research, different youtube tutorials and documentations has been followed. The findings has been described below.

AWS is a fully managed and continuously trained NLP service. In order to use AWS’s image recognition service, the user will have to create an account in AWS and setup the appropriate service, S3 bucket for storage, cloud watch for cloud notification and monitoring, and SNS to trigger an email notification as soon as an image has been detected.

The image detection gets done via the Lambda function which uses python as its primary language. For image recognition for an example, there is a library called Boto3. It has different function written for convenience. All the user has to do is select the right function and apply it to the Lambda function. The link to the documentation/library has been provided below.

https://boto3.amazonaws.com/v1/documentation/api/latest/reference/services/sns.html#SNS.Client.publish

When it comes to recognizing the text from an image or anything, AWS uses Natural language process. This service is known as Amazon Comprehend. This service analyzes the unstructured text data using NLP. This allows the user to perform tasks like language detection which is capable of detecting up to 100 languages. One of the use cases in DHL is that when a uses is experiencing a virtual tour, and he/she come across a sign in a different language and would like to know what it means, he/she will be able to use the Amazon Comprehend service to run a analysis on that language.

Costs

AWS allows the user to have a free account with some limitations. The account stays free for 12 months and the Image rekognition service does not cost anything. However, it allows the user to be able to analyze only 5000 images per month and store only 1000 pieces of face metadata per month.
