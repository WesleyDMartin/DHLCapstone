Summary

Outline the reasoning and design of the prototype, its current status, and the results, once found

The initial prototype for our project is going to be a slimmed down, but functional version of the complete project. We plan to implement a text based version of our Conversational Interface Research, and a strictly 2D, non immersive version of the video question responses.

Prototype Purpose

The purpose of the prototype is not to represent a version of the final version, at least not now. Rather, this prototypes main purpose is to test, compare, and contrast the APIs of the platforms we are considering for the final product. The information regarding that research and decision making can be found in Initial Research.

The prototype will be a C# application which will be able to generalize the calls to each API endpoint, making image/text analysis requests with identical images, parsing the JSON results, and then comparing the outputs.

As we will be developing the test application in C#, it makes sense to track the various API libraries that will be able to run natively in C# and communicate with the endpoints.
